TFcmf是一款基于ThinkPHP5+Bootstrap+layui的极速后台开发框架。
===============

## **主要特性**

* 基于`Auth`验证的权限管理系统
* API快速开发，已完成初始化程序
* 完善的前端功能组件开发
    * 基于`AdminLTE` `layui`开发
    * 基于`Bootstrap`开发，自适应手机、平板、PC 
    * 封装了部分方法，开发快速简单，没有文档，看例子


## **安装使用**

建议git克隆安装,会经常更新功能和完善。数据库文件在data目录下。
